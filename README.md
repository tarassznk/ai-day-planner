# AI Планер дня

Мобільний веб-планер: вивали все з голови голосом або текстом — AI (Claude)
перетворює хаос на структуровані задачі й план на сьогодні.

**Наскрізний сценарій:** Brain dump → AI аналізує текст → створює задачі
(пріоритет, час, дедлайн, теги) → формує план на сьогодні.

## Можливості

- 🎙️ **Голос і текст** — диктуєш або пишеш усе підряд (Web Speech API, `uk-UA`)
- 🤖 **AI-парсинг** — Claude Haiku розбиває потік думок на атомарні задачі,
  ставить пріоритет (P1–P4), оцінює час, витягує дедлайни й додає розумні теги
- 📥 **Inbox → Today** — AI сам пропонує, що зробити сьогодні
- 🔴 **Нагадування про P1** — не даємо забути пріоритетні задачі
- ⚖️ **Реалістичність плану** — попереджає, якщо задач більше, ніж вміщує день
- ↪️ **Перенос на завтра** — невиконане не губиться
- 📱 **Mobile-first** — проходиться однією рукою, темна тема

## Стек

- **Next.js (App Router)** + React + TypeScript
- **Anthropic API** (`claude-haiku-4-5`) — виклик тільки на сервері
- **localStorage** — задачі зберігаються у браузері (без логіну)
- Деплой: **Vercel**

## Локальний запуск

```bash
npm install
# створи .env.local зі своїм ключем:
#   ANTHROPIC_API_KEY=sk-ant-...
npm run dev
```

Відкрий http://localhost:3000

## Змінні оточення

| Змінна              | Опис                                              |
| ------------------- | ------------------------------------------------- |
| `ANTHROPIC_API_KEY` | Ключ з console.anthropic.com. Тільки на сервері.  |

На Vercel цю змінну додають у **Project Settings → Environment Variables**.
Ключ ніколи не потрапляє в код браузера чи git.

## Безпека

- API-ключ живе лише в серверному route `app/api/parse/route.ts` (env-змінна).
- `.env.local` у git не комітиться (див. `.gitignore`).
